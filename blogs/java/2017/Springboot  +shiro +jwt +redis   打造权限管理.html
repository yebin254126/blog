# Springboot +shiro +jwt +redis 打造权限管理模块

  1. # 环境准备

    • 使用Maven管理jar依赖

      我这里使用的Springboot 2.x ,数据层框架使用的是Mybatis，连接池使用的是阿里的Druid连接池，maven依赖如下:

     <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-web</artifactId>
        </dependency>
        <dependency>
            <groupId>org.mybatis.spring.boot</groupId>
            <artifactId>mybatis-spring-boot-starter</artifactId>
            <version>2.1.1</version>
        </dependency>

        <!-- https://mvnrepository.com/artifact/org.springframework.boot/spring-boot-starter-		data-redis -->；
        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-data-redis</artifactId>
            <version>2.2.1.RELEASE</version>
        </dependency>
        <dependency>
            <groupId>mysql</groupId>
            <artifactId>mysql-connector-java</artifactId>
            <scope>runtime</scope>
        </dependency>
        <dependency>
            <groupId>com.alibaba</groupId>
            <artifactId>druid-spring-boot-starter</artifactId>
            <version>1.1.10</version>
        </dependency>
        <dependency>
            <groupId>org.projectlombok</groupId>
            <artifactId>lombok</artifactId>
            <optional>true</optional>
        </dependency>
        <dependency>
            <groupId>org.apache.shiro</groupId>
            <artifactId>shiro-spring-boot-starter</artifactId>
            <version>1.4.0</version>
        </dependency>
        <!-- https://mvnrepository.com/artifact/com.auth0/java-jwt -->
        <dependency>
            <groupId>com.auth0</groupId>
            <artifactId>java-jwt</artifactId>
            <version>3.8.3</version>
        </dependency>

    • 项目结构配置

      ​ 配置数据源及相关配置

      ​

      spring.datasource.type=com.alibaba.druid.pool.DruidDataSource
spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver
spring.datasource.url=jdbc:mysql://localhost:3306/shiro?useSSL=false&serverTimezone=UTC&characterEncoding=utf8
spring.datasource.username=xxx
spring.datasource.password=xxxx

#mybatis
mybatis.type-aliases-package=com.yebin.jwtshiro.domain
mybatis.mapper-locations=classpath:mapper/*.xml


#显示mapper查询  日志级别
logging.level.com.yebin.jwtshiro=debug
logging.level.com.yebin.jwtshiro.mapper=debug
#保存日志到文件中去
logging.file.path=log
logging.file.name=spring.log



#redis 数据源

spring.redis.host=localhost
spring.redis.port=6379
spring.redis.password=xxxx
